## ğŸš€ Overview

**InspireNest** is an idea-driven web application created to provide a **safe, engaging, and supportive digital environment** for intellectual disability children. The platform focuses on improving **IQ, EQ, emotional well-being, creativity, and daily self-care skills**, while also enabling parents and therapists to monitor progress effectively.

The project was developed efficiently by using the right tools and workflows to quickly transform the idea into a functional frontend, allowing greater focus on **design quality, usability, accessibility, and user experience**.
